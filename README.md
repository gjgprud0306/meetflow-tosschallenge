# MeetFlow Web Prototype

토스 2026 디자이너 챌린지 제출용 MeetFlow 웹 프로토타입입니다.

## Stack

- React
- TypeScript
- Vite
- React Router
- Tailwind CSS
- shadcn/ui
- Desktop 1440px 기준

## Structure

- `src/routes.tsx`: route definition
- `src/layouts`: route layouts
- `src/pages`: page-level entries
- `src/components/ui`: shadcn/ui components
- `src/lib`: shared utilities
- `src/mocks`: mock data
- `src/types`: shared TypeScript types
- `src/styles/globals.css`: Tailwind and shadcn/ui tokens

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
